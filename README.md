# Coffee Sales Performance Dashboard

## Project Overview
This project features an interactive Excel dashboard designed to track and analyze sales performance for a global coffee retailer. It provides insights into revenue trends, regional performance, and customer loyalty.

## Key Features & Technical Skills
* **Data Modeling:** Utilized **XLOOKUP** and **INDEX/MATCH** for dynamic data retrieval and cross-referencing across datasets.
* **Interactivity:** Integrated **Slicers** for real-time filtering by Roast Type, Package Size, and Loyalty Card status.
* **Time-Series Analysis:** Developed a "Total Sales Over Time" line chart to identify growth patterns and seasonality.
* **Geospatial & Customer Insights:** Visualized sales by country and identified the "Top 5 Customers" to support strategic marketing.

## How to Use
1. Download the `Coffee_Sales_Intelligence_Dashboard.xlsx` file.
2. Open in Microsoft Excel (Enable Macros/Content if prompted).
3. Use the slicers on the right to interact with the data visualizations.
